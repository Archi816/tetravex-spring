# Tetravex

Browser-based implementation of the classic Tetravex puzzle game, 
built with Spring Boot.

## Gameplay

Place tiles on the board so that all touching edges match numbers.
The game ends when the puzzle is correctly solved.

## Difficulty Levels

| Level | Grid |
|-------|------|
| Easy | 3×3 |
| Medium | 4×4 |
| Hard | 6×6 |
| Impossible | 10×10 |

## Features

- Multiple difficulty levels
- Score tracking
- Rating system
- Comments system
- Console UI + REST API backend

## Tech Stack

Java · Spring Boot · REST API

## Run

```bash
./mvnw spring-boot:run
```
